# 素材收集 material-collector

通过四轮渐进式网页搜索，系统性收集与乐活业务相关的热门素材。

## 业务背景

本技能服务于 iLohhas乐活 的内容创作，核心话题：
- **左右脑管理哲学**：陶老师原创，区别于其他OKR服务的核心差异化
- **OKR**：以左右脑思想为基础的目标管理体系
- **组织进化**：构建支撑双脑的组织体系（文化、人才、结构、环境）
- **乐活人生**：幸福公式与个人成长方法论
- **阶段理论**：0-0.1探索期、0.1-1突破期、1-10成长期、10-100成熟期

详见 `references/user-profile.md`

## 平台搜索策略

| 平台 | 搜索方式 | 覆盖内容 |
|------|---------|---------|
| 微信公众号 | `site:mp.weixin.qq.com {keyword}` | 公众号热文、管理类文章 |
| 百度 | `{keyword}` (Baidu搜索) | 中文资讯、案例、讨论 |
| 谷歌 | `{keyword}` (Google搜索) | 全球视角、学术研究、海外案例 |

## 话题扩展规则

以用户提供的 `keyword` 为核心，根据乐活话题领域扩展2-5个变体：

**扩展方向：**
- 左右脑管理 → 搜索"左脑管理"、"右脑管理"、"左右脑OKR"
- OKR相关 → 搜索"OKR实施"、"OKR案例"、"KPI对比OKR"
- 组织进化 → 搜索"组织变革"、"组织文化"、"敏捷组织"
- 个人成长 → 搜索"知识型员工"、"个人OKR"、"目标管理"

保持约20%探索性查询（跨领域/相邻关键词），避免信息茧房。

## 四轮搜索流程

### 第一轮：权威来源

目标：获取一手、权威信息。

```
# 微信生态
site:mp.weixin.qq.com {keyword}
site:weixin.qq.com {keyword}

# 官方/权威
{keyword} 官方公告
{keyword} GitHub
{keyword} official announcement
```

**重点关注**：管理类公众号热文、OKR工具官方博客、知名企业案例

### 第二轮：深度解析

目标：理解机制原理，获取详细介绍。

```
{keyword} 详细介绍
{keyword} 教程 tutorial
{keyword} how it works
{keyword} 深度解读
{keyword} 案例 case study
```

### 第三轮：对比评测

目标：获取多元视角和对比分析。

```
{keyword} vs {竞品/替代方案}
{keyword} 评测 review
{keyword} 优缺点
{keyword} pros cons
```

### 第四轮：补充验证

目标：填补信息空白，获取最新动态。

```
# 分析前三轮缺口
missing_info = analyze_gaps(previous_results)

# 缺口补充
{keyword} 最新 2025 2026
{keyword} 趋势 trend
{keyword} 研究报告
```

## 去重与聚类

- **标题完全相同** → 保留最高热度来源
- **同一事件多篇报道** → 保留最早来源 + 最高热度各一条
- 按话题粗略分组，打上话题标签（如"左右脑OKR"、"组织文化"、"个人成长"）

## Optional State（可选，正负样本参考）

若 `material-pool/state/liked_topics.json` 存在，读取高频关键词/实体用于扩展查询方向：
- 高频实体：优先搜索
- 高频话题：优先聚类

若 `material-pool/state/rejected_topics.json` 存在，对候选话题做相似度预检：
```bash
python3 ../material-pool/scripts/material_pool_state.py similarity --text "{候选话题}" --against rejected --topk 3
```
若最高相似度 ≥ 0.85，该候选降权处理（降低质量评分0.5分）。

若状态文件不存在，正常执行。

## 执行步骤

1. **接收 keyword** — 用户输入的核心主题
2. **加载业务上下文** — 读取 `references/user-profile.md` 中的核心话题领域
3. **读取正样本库（可选）** — `material-pool/state/liked_topics.json`，提取高频关键词辅助扩展
4. **扩展查询** — 按话题领域生成2-5个变体，保持20%探索性
5. **执行第一轮** 搜索（权威来源）
6. **执行第二轮** 搜索（深度解析）
7. **执行第三轮** 搜索（对比评测）
8. **分析缺口** — 识别前三轮未覆盖的信息
9. **执行第四轮** 搜索（补充验证）
10. **负样本预检（可选）** — 相似度≥0.85的候选降权
11. **去重聚类** — 合并重复，分组打标签
12. **生成Hook候选** — 每条素材生成5个候选钩子
13. **生成报告** — 结构化输出 + MATERIALS_JSON
14. **报告摘要** — 给用户

## 平台搜索技巧

### 微信公众号
- 使用 `site:mp.weixin.qq.com` 限定来源
- 重点关注：管理类、创业类、人力资源类公众号
- 热门文章通常在搜索结果前排

### 百度
- 重点关注「资讯」和「最新」标签
- 百度指数可辅助判断热度趋势
- 搜索结果分类：资讯、网页、贴吧、视频

### 谷歌
- `site:zhihu.com` 搜知乎深度讨论
- `site:36kr.com` 搜科技媒体
- `site:linkedin.com` 搜海外管理视角
- 时间筛选器限定最近一年

## 质量评分参考

- **8-10分**：权威来源 + 独家数据 + 深度分析
- **5-7分**：主流媒体 + 二手解读 + 事实陈述
- **3-4分**：自媒体 + 主观观点 + 信息量少
- **1-2分**：低可信度来源或重复内容
