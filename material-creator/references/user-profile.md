# 乐活内容定位

此文件定义内容创作的核心方向，所有 material-* skills 均应参考此配置。

## 品牌信息

- **品牌**：iLohhas乐活 / 乐活OKR
- **使命**：点亮工作，让工作点亮人生
- **价值观**：认真、乐观、不怕失败
- **slogan**：有目标、认真干、去执念

## 核心话题领域

| 话题 | 说明 | 关键词举例 |
|------|------|-----------|
| 左右脑管理哲学 | 陶老师原创管理思想，核心差异化 | 左脑、右脑、理性、感性、ROI、迭代速度、左右脑OKR |
| OKR | 目标管理工具，与左右脑结合 | 目标管理、OKR立盘、OKR复盘、承诺型OKR、挑战性OKR |
| 组织进化 | 帮助企业构建支撑双脑的组织体系 | 组织文化、组织人才、组织结构、组织环境、创新力、组织力 |
| 乐活人生 | 幸福公式与个人成长 | 有目标、认真干、去执念、干一行爱一行 |
| 创业与阶段 | 企业/个人的0-0.1到10-100四阶段 | 探索期、突破期、成长期、成熟期 |
| 管理新思维 | 面向AI时代的管理方法 | 知识型组织、分布式管理、个体AI化、超级个体 |

## 目标受众

1. **首要**：企业管理者、创业者（面临组织挑战）
2. **次要**：知识型组织中的高算力个体（自我管理者）
3. **补充**：对管理感兴趣的年轻人（乐活青创营群体）

## 内容分发渠道

- 视频号：【乐活范】【学管理】栏目
- 小红书：乐活OKR账号
- 线下：乐活大学（培训课堂）

## 内容风格

| 风格 | 适用场景 | 特点 |
|------|---------|------|
| 高价值干货 | 教程、方法论、工具推荐 | 数字开头、清单结构、可收藏 |
| 深度解析 | 原理讲解、案例分析 | 左右脑视角、阶段性分析 |
| 热点评论 | 新闻解读、事件点评 | 快速反应、独特角度 |
| 故事洞察 | 案例复盘、个人经历 | 具体场景+转折+金句 |
| 犀利观点 | 行业评论、反常识 | 有立场、有态度、能引发讨论 |

## 创作人设

- 认真：不敷衍，每篇有真观点
- 乐观：积极但不做作，不鸡汤
- 不怕失败：有挑战精神，鼓励探索
- 专业但不装：用说人话的方式讲管理
