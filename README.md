# Lohhas Skills

> 乐活 Skills —— 让 AI 助手更强大

## 📦 包含 Skills

### 1. spf-unified-workflow
统一工作流技能，包含持久化记忆、TDD 执行、15 个生产力和开发技能。

**功能：**
- 持久化 AI 记忆循环
- 动态 TDD 开发流程
- 多 Agent 协作
- 代码审查工作流

**包含子技能：**
| 技能 | 用途 |
|------|------|
| spf-plan | 创建实施计划 |
| spf-exec | TDD 执行 |
| test-driven-development | TDD 工作流 |
| systematic-debugging | 系统调试 |
| brainstorming | 头脑风暴 |
| dispatching-parallel-agents | 多 Agent 调度 |
| writing-skills | 创建新技能 |

## 🚀 快速开始

### 方法一：克隆到 OpenClaw Skills 目录

```bash
# 克隆仓库
git clone https://github.com/Heypyyyyf/lohhas_skills.git

# 复制到 OpenClaw skills 目录
cp -r lohhas_skills/* ~/.openclaw/skills/

# 重启 OpenClaw
openclaw restart
```

### 方法二：使用 ClawHub（待发布）

```bash
clawhub install lohhas-skills
```
