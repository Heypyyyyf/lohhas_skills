# 分镜设计规范

## 景别定义与适用场景

| 景别 | 取景范围 | 适用场景 | 情感强度 |
|------|----------|----------|----------|
| 远景(WS) | 全貌+环境+人物 | 建立场景、气势、广阔 | 弱 |
| 全景(WS) | 完整人物+脚部 | 展示动作全貌、人物与环境关系 | 中 |
| 中景(MS) | 膝盖以上 | 日常对话、正常叙事 | 中 |
| 近景(CU) | 胸口以上 | 强调表情、情感交流 | 强 |
| 特写(ECU) | 脸部/局部细节 | 强调重点、情绪爆发 | 极强 |

### 景别选择逻辑

```
叙事为主 → 中景/全景
情绪为主 → 近景/特写
建立氛围 → 远景/全景
转场过渡 → 远景承接特写
```

## 运镜方式详解

| 运镜 | 效果 | 速度建议 | 适用场景 |
|------|------|----------|----------|
| 推镜头 | 走近、聚焦、强调、紧张 | 慢推(3-5s) | 发现细节、聚焦情绪 |
| 拉镜头 | 远离、揭示、放松、孤独 | 慢拉(3-5s) | 揭示环境、释放情绪 |
| 摇镜头 | 环顾、扫描、展示环境 | 中速(2-4s) | 介绍场景、寻找主体 |
| 移镜头 | 跟随、沉浸、穿越 | 与主体同步 | 跟随行动、展示空间 |
| 跟镜头 | 跟随主体、强调关系 | 与主体同步 | 叙事连续、人物追踪 |
| 固定镜头 | 稳定、观察、强调内容 | 静止 | 静谧感、强调台词 |
| 升降镜头 | 变化视角、揭示层次 | 慢速 | 展示规模、营造氛围 |
| 环绕镜头 | 围绕主体、强调重要性 | 中速 | 强调、戏剧化 |

### 运镜组合示例

```
远景 + 固定 → 平静开场
中景 + 推 → 逐渐聚焦
特写 + 拉 → 情绪释放
全景 + 摇 → 环顾环境
```

## 角度与透视

| 角度 | 效果 | 适用场景 |
|------|------|----------|
| 平视 | 平等、客观、亲近 | 日常叙事、对话 |
| 仰拍 | 威严、力量、崇高 | 英雄登场、强调地位 |
| 俯拍 | 渺小、无力、压抑 | 展示全局、情绪低落 |
| 侧拍 | 立体、客观、中立 | 侧面观察、两人关系 |
| 顶拍 | 抽象、图案化、独特视角 | 艺术感、舞蹈/运动 |
| 荷兰角 | 失衡、紧张、动荡 | 冲突、困惑、非常态 |

## 光线设计

### 光线类型

| 类型 | 描述 | 情绪 |
|------|------|------|
| 自然光 | 阳光、窗光 | 真实、自然 |
| 柔光 | 散射光、阴天 | 温柔、梦幻 |
| 硬光 | 直射、强烈 | 紧张、硬朗 |
| 暖光 | 黄色/橙色调 | 温暖、怀旧 |
| 冷光 | 蓝色调 | 冷静、科技、未来 |
| 戏剧光 | 明暗对比强烈 | 戏剧性、电影感 |
| 伦勃朗光 | 三角光效 | 立体、艺术感 |

### 光线方向

| 方向 | 效果 |
|------|------|
| 正面光 | 明亮、平面、亲切 |
| 侧光 | 立体、明暗对比 |
| 逆光 | 轮廓、剪影、浪漫 |
| 顶光 | 舞台感、突出中心 |
| 底光 | 恐怖、神秘、不自然 |

## 构图原则

### 三分法
- 主体放在四条线或四个交点上
- 视线方向留白
- 运动方向前方留白

### 前景/中景/背景
- 三层纵深增加立体感
- 前景可做框架构图
- 背景要干净不杂乱

### 视线高度
- 眼平线：平等视角
- 腰部以下：俯视
- 头顶以上：仰视

## 衔接进阶技巧

### 动势衔接
```
人物A向左看 → 下个镜头主体应在左侧
人物站起 → 下个镜头可以是走/站
视线方向 → 下个镜头接视线终点
```

### 视线匹配（180°法则）
```
看信（特写） → 信的内容（全屏文字）
看窗外 → 窗外的风景
打电话 → 对方听筒/表情
```

### 声音衔接
```
前一镜头结尾音 → 后一镜头开始音（音效延续）
同期声提前1秒 → 视觉延迟
音乐卡点 → 动作/转场与节拍吻合
```

## 转场详解

### 硬切适用
- 话题切换
- 时间跳跃
- 快节奏剪辑
- 反差场景
- 强调对比

### 叠化适用
- 连续动作（1+1=1）
- 回忆片段
- 情绪延续
- 缓慢过渡
- 梦境感

### 淡入淡出适用
- 段落分隔
- 情绪转换
- 开头/结尾
- 时间流逝
- 日出日落

### 特殊转场
- **遮挡转场**：物体挡镜头后换场景（手挡/门挡/车挡）
- **旋转转场**：场景旋转中变换
- **缩放转场**：从特写拉到远景变换场景
- **色调切换**：冷暖色调转换表示情绪变化
- **时间分裂**：同一位置不同时空的切换
- **声音转场**：用对白/音乐过渡

## 5秒片段的内容密度

5秒 ≈ 3-5个镜头切换
5秒 ≈ 30-50字旁白
5秒 ≈ 1个主要动作完成

### 5秒画面 Checklist

```
✓ 场景是否建立清楚
✓ 主体的表情/动作是否清晰
✓ 光线氛围是否符合情绪
✓ 镜头运动是否有意义（不是炫技）
✓ 台词是否精简有力
✓ 和前后片段是否衔接
✓ 总描述字数是否≥400字
```
