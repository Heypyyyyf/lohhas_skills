# video-prompt-generator

创作池选题 → 1分钟视频分镜提示词（20个5秒片段）。

**质量标准：每个片段描述 ≥400字，确保画面感强、可执行性高。**

## 核心概念

| 概念 | 说明 |
|------|------|
| 片段（Shot） | 5秒的最小内容单元 |
| 分镜（Storyboard） | 20个片段的有序组合 |
| 衔接（Transition） | 片段之间的过渡逻辑 |

## 执行步骤

```
接收选题（从创作池获取）
  ↓
读取创作内容（从飞书多维表格获取）
  ↓
分析核心主线（1个核心观点）
  ↓
拆解内容节奏（起-承-转-合）
  ↓
分配20个片段的时间结构
  ↓
逐个生成片段内容（每段≥400字）
  ↓
衔接检查与优化
  ↓
输出完整分镜脚本
```

### Step 1：获取创作池选题

1. 用 `feishu_bitable_app(action=list)` 找到名为"创作池"的多维表格
2. 用 `feishu_bitable_app_table(action=list)` 获取数据表 table_id
3. 用 `feishu_bitable_app_table_record(action=list)` 获取记录
4. 按"入选时间"降序排序，取第一条记录
5. 提取关键字段：
   - 标题（标题）
   - 摘要（摘要）
   - 创作角度（创作角度）
   - 推荐风格（推荐风格）
   - 完整内容（创作内容）

### Step 2：分析选题

提取以下信息：
- **核心主线**：一句话概括视频要传达的核心观点
- **内容类型**：热点评论/高价值干货/深度解析/故事洞察
- **目标受众**：谁在看
- **情绪基调**：理性分析/情感共鸣/犀利点评

### Step 3：分配时间结构

| 段落 | 片段范围 | 时长 | 功能 | 占比 |
|------|----------|------|------|------|
| 开场Hook | 01-02 | 10s | 抓住注意力 | 16.7% |
| 引入主题 | 03-05 | 15s | 交代背景/问题 | 25% |
| 核心展开 | 06-14 | 45s | 重点内容 | 45% |
| 高潮/转折 | 15-18 | 20s | 情绪高点 | 33.3% |
| 收尾CTA | 19-20 | 10s | 行动召唤/金句结尾 | 16.7% |

### Step 4：逐个生成片段

每个片段包含以下板块（每片段≥400字）：

```yaml
片段 01 / 20
时间码: 00:00 - 00:05
类型: 开场Hook

## 画面描述（≥150字）
【场景】[具体描述：位置、光线、时间、氛围]
【主体】[人物外貌特征、表情神态、衣着打扮]
【动作/动态】[具体的动作描写，动作的起始和结束状态]
【环境细节】[背景中的重要元素]

## 镜头语言（≥100字）
【景别】[远景/全景/中景/近景/特写选择理由]
【运镜方式】[推/拉/摇/移/跟/固定+具体描述]
【角度】[平视/仰拍/俯拍/侧拍+创意角度]
【焦距/景深】[广角/标准/长焦]

## 台词/旁白（30-50字）
[该片段的台词，若无则写"无"]
[语速、语调建议]

## 音效/配乐（≥50字）
【背景音乐】[音乐风格、节奏、情绪]
【音效设计】[环境音、动作音效、强调音效]

## 氛围/情绪（≥50字）
[整体氛围描述]
[情绪走向：平静→紧张/低落→高潮等]

## 转场设计（≥50字）
[本片段结尾动作]
[下一个片段的过渡方式：硬切/叠化/淡出等]
[过渡的理由]
```

## 分镜衔接原则

### 景别衔接规则

```
远景 → 全景 → 中景 → 近景 → 特写（递进，增强情绪）
特写 → 远景（拉伸，释放情绪或转场）
同景别跳切需避免（除非刻意为之）
```

### 动作衔接规则

```
主体移动方向要连贯
视线方向要匹配
避免180°跳切
动势要承接（上镜头的手，下镜头继续这个动作）
```

### 转场方式库

| 转场 | 适用场景 | 时长 |
|------|----------|------|
| 硬切 | 快节奏、切换话题 | 瞬时 |
| 叠化 | 回忆、连续动作 | 1-2秒 |
| 淡入淡出 | 段落分隔、情绪过渡 | 1-2秒 |
| 滑动/推拉 | 强调重点、转场 | 1-2秒 |
| 跳转 | 打破第四墙、幽默效果 | 瞬时 |
| 遮挡转场 | 物体挡镜头 | 瞬时 |
| 旋转转场 | 场景旋转 | 1-2秒 |
